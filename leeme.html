Vistas que componen esta entrega

  • login - Página de Login, se necesita un usuario y contraseña para entrar, o Regisrarse.
  • registro - Permite al usuario registrarse para iniciar sesión.
  • sin_registro - Vista de la web sin estar logueado. Muestra actividades públicas, creadas por otros usuarios.
  • home - Página de inicio que tiene una vista general de las funcionalidades de GoOut.
  • crear - Permite crear una nueva actividad. Especificando varias opciones.
  • mis_actividades - Pagina donde el usuario puede consultar sus actividades recientes e Historial de actividades pasadas.
  • buscar - Vista para buscar una actividad concreta.
  • actividad -(solo visible cuando pinchas en una actividad anteriormente listada) Contiene detalles de una actividad y un foro que pueden hacer comentarios los participantes de la misma.
  • perfil - (solo accesible cuando pienchas en la foto de perfil) Muestra los datos de un usuario, así como las actividades a las que asistirá.
